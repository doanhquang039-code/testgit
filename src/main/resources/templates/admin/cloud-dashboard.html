Cấu hình AWS S3:
  1. Tạo S3 bucket trên AWS Console
  2. Tạo IAM user với quyền S3
  3. Set aws.s3.enabled=true
  4. Set aws.s3.access-key và aws.s3.secret-key
  5. Set aws.s3.bucket=your-bucket-name
Hướng dẫn cấu hình
🔔 Firebase FCM
1. Tạo project console.firebase.google.com
2. Download service account Project Settings → Service Accounts
3. Đặt file JSON vào src/main/resources/
4. Set config firebase.enabled=true
📁 Google Drive
1. Enable Drive API console.cloud.google.com
2. Tạo Service Account IAM → Service Accounts
3. Share folder với SA email Google Drive → Share
4. Set config google.drive.enabled=true
