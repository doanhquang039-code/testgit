## 📝 Build Commands Reference

### Flutter
```bash
# Clean
flutter clean

# Get dependencies
flutter pub get

# Build Android APK
flutter build apk --release

# Build Web
flutter build web --release

# Build iOS (macOS only)
flutter build ios --release

# Build Windows
flutter build windows --release
```

### Node.js
```bash
# Install dependencies
npm install

# Start development
npm run dev

# Start production
npm start
```

### ASP.NET Core
```bash
# Restore packages
dotnet restore

# Build
dotnet build

# Run
dotnet run

# Publish
dotnet publish -c Release
```

### Spring Boot (when Maven is installed)
```bash
# Clean and build
mvn clean package

# Skip tests
mvn clean package -DskipTests

# Run
mvn spring-boot:run
```
